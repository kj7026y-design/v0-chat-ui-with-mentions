# 채팅 명령어 QA/AC

## 자동화 AC

1. `/휴대폰`은 시간, 무음, HD, 5G, 안테나, 배터리가 포함된 상태 표시줄을 출력한다.
2. 최근 통화·문자·브라우저 검색·유튜브 시청·결제·실행 앱을 모두 휴대폰 목록 형태로 출력한다.
3. 연락처 역할, 사용자 이름, 검색어와 기록 내용은 작품 설정·현재 상태·최근 대화에서 가져온다.
4. 같은 대화에서 연속 실행해도 시간 차이, 기록 항목, 금액과 표현이 달라진다.
5. SNS·상태창·시청자 반응·요약도 고정 더미 문구가 아니라 현재 장면을 반영한다.
6. 자동 명령어 예상 길이를 생성 전에 반영하고, AI 답변과 실제 보조 메시지의 합계를 1500자 이하로 맞춘다.
7. AI 700자를 포함할 공간이 있으면 자동 명령어가 있어도 최소 답변 길이를 700자로 유지한다.
8. 재생성 시 같은 턴의 명령어 카드도 새 답변 문맥으로 다시 만들고 기존 메시지 ID를 유지한다.

실행 명령:

```powershell
npx tsx --test tests/chat-commands.test.ts tests/chat-interactions.test.ts
```

## 수동 통합 QA

- 일반 실행: 최근 대화 뒤 `/휴대폰`을 두 번 실행해 두 화면의 기록이 다르고 현재 인물·장면을 반영하는지 확인한다.
- 자동 명령: 휴대폰과 SNS를 자동 명령으로 선택하고 답변을 생성해 AI 답변과 두 카드가 같은 턴으로 저장되는지 확인한다.
- 자동 진행: 입력 없이 자동 진행했을 때 생성된 새 장면을 휴대폰·SNS 기록이 반영하는지 확인한다.
- 재생성: 자동 명령이 붙은 AI 답변을 재생성해 AI 답변과 같은 턴의 명령 카드가 함께 갱신되는지 확인한다.
- 복원: 페이지를 새로고침해 저장된 휴대폰 메시지가 휴대폰 프레임 스타일로 다시 표시되는지 확인한다.
